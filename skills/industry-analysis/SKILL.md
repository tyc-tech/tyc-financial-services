# 行业分析辅助

## 触发条件

行业研究员撰写行业报告、咨询公司交付行业研究、企业战略部做行业赛道扫描时触发。

关键词：行业研究、产业图谱、行业格局、头部厂商、赛道分析

## 输入要求

- **industry** (必填): 行业关键词（如"储能电池"、"工业机器人"）
- **region** (可选): 地域范围
- **timeRange** (可选): 时间窗口（默认近 3 年）

## 执行流程

### Step 1: 行业参与者扫描
- `search_companies_by_industry_region` — 行业地域内企业清单
- `search_companies_by_ranking` — 行业榜单对齐

### Step 2: 头部厂商画像（TOP 20）
对头部厂商并发调用：
- `get_company_registration_info` — 基础
- `get_company_scale` — 规模
- `get_patent_info` — 专利规模
- `get_ipr_score` — 创新力
- `get_bidding_info` — 中标记录（B 端厂商重要）
- `get_qualifications` — 资质等级

### Step 3: 行业需求侧（招投标侧）
- `search_bids` — 行业关键词招投标搜索
- 按甲方/乙方聚合，分析行业总盘子与季度趋势

### Step 4: 行业资本侧
- `get_invest_agency_funds` — 行业 TOP 投资机构管理基金
- 按投资机构 → 被投企业反向挖掘行业 cap table

### Step 5: 行业专利壁垒
- `search_patents` — 行业专利数量/类型分布
- 识别专利聚集企业

## 错误处理

- `search_bids` 无结果 → 该 Step 标注"行业公开招标数据不足"
- 头部名单不足 20 → 按实际数量输出，注明样本量
- 部分企业专利不全 → 该企业专利字段标注 `[~]`
