# 上市公司财务深度分析（TYC 独有扩展）

## 触发条件

二级市场投资研究、Pre-IPO 估值参考、上市公司收购前财务尽调、价值投资入场分析。

关键词：上市公司财务、三表分析、ROE 拆解、股东结构、Pre-IPO 估值

## 输入要求

- **searchKey** (必填): 上市公司名称、股票代码 或 信用代码

## 执行流程

### Step 1: 上市信息确认
- `get_listing_info` (`searchKey`) — 股票代码 / 上市日期 / 总市值

### Step 2: 财务三表
- `get_income_statement` — 利润表
- `get_balance_sheet` — 资产负债表
- `get_cash_flow_statement` — 现金流量表

### Step 3: 财务指标
- `get_financial_summary` — 财务简析（净利率、毛利率、ROA、ROE 等）
- `get_financial_main_indicators` — 年度 + 季度主要指标（每股收益等）

### Step 4: 股权结构
- `get_share_structure` — 股本结构 + 变动 + 分红 + 配股
- `get_stock_shareholders` — 十大股东 + 流通股东 + 参股控股

### Step 5: 董监高与公司治理
- `get_stock_executives` — 董监高 + 重要人员 + 联系方式
- `get_stock_violations` — 违规处理

### Step 6: IPO 历史
- `get_stock_prospectus` — 招股书

- 非上市公司 → 提示"该企业未上市，建议使用 `tyc-financial-data` 看年报财务"
