# 供应链风险评估（投资视角）

## 触发条件

投资尽调阶段评估目标公司的供应链韧性、制造业投资项目侧写、产业链投资策略分析时触发。

关键词：供应链风险、上下游、集中度、韧性、单一客户依赖、卡脖子

> **与 `/tyc-supply-risk`（采购仓 supply）的区别**：
> - 本 Skill 面向**投资人**，5 步短链，30 秒输出，决策快速 go/no-go
> - `/tyc-supply-risk` 面向**采购总监**，18 类风险长链，深度供应商管理

## 输入要求

- **searchKey** (必填): 目标企业名称 或 信用代码

## 执行流程

### Step 1: 上下游关系识别
- `get_suppliers_and_customers` — 直接上下游（供应商 + 客户）
- 识别 TOP 5 供应商 & TOP 5 客户

### Step 2: 集中度测算
对 TOP 上下游对手方并发调用：
- `get_company_registration_info` — 基础
- `get_risk_overview` — 风险

### Step 3: 海关 / 跨境供应链
- `get_import_export_credit` — 进出口信用（是否涉及跨境 / 断供风险）

### Step 4: 需求侧（如目标为制造业）
- `get_bidding_info` — 目标公司中标记录（B 端厂商的客户集中度）

### Step 5: 整体供应链风险
- `get_risk_overview` — 本身风险（反向判断是否会拖累上下游）

## 输出格式

```markdown
# 供应链风险评估（投资视角）— {name}

> 出具: {ISO8601} · 短链版 · 适用投资 go/no-go

## 一、上下游概览

### 供应商端
- 识别的主要供应商: {n} 家
- TOP 5 供应商: ...
- 供应商集中度（TOP 3 占比）: {ratio}
- 集中度评价: 高 / 中 / 低

### 客户端
- 识别的主要客户: {n} 家
- TOP 5 客户: ...
- 客户集中度（TOP 3 占比）: {ratio}
- 单一客户依赖风险: 有 / 无

## 二、跨境供应链
- 海关信用等级: {grade}
- 进出口业务占比: ... （若可推断）
- 跨境断供风险: 高 / 中 / 低

## 三、对手方风险
- 供应商有重大风险: {n} 家
- 客户有重大风险: {n} 家
- 具体关注: ...

## 四、中标集中度（B 端厂商）
- 近 3 年中标次数: {n}
- 甲方集中度（TOP 3 占比）: {ratio}
- 政府采购 / 大客户占比: ...

## 五、投资结论

### 供应链韧性: 强 / 中 / 弱

| 维度 | 评价 |
|------|------|
| 供应商替代性 | 易 / 中 / 难 |
| 客户分散度 | 高 / 中 / 低 |
| 跨境风险 | 低 / 中 / 高 |
| 对手方稳健性 | 强 / 中 / 弱 |

### 投资影响
- 估值建议: 正常 / 折价 {ratio}（供应链脆弱性）
- 重点 DD: 核心供应商替代方案、单一客户合同条款
- 交割前条件: 如存在供应商单点风险，建议分散 / 备选

## 六、建议追加深度 DD
若发现高风险 → 调用采购仓 `/tyc-supply-risk` 深度评估
```

## 错误处理

- `get_suppliers_and_customers` 无数据 → 提示"该企业上下游关系未公开，建议结合企业自披露"
- 全部指标 `_empty` → 仍输出"供应链公开数据不足"结论
- 5 步中任一步失败 → 不影响其他步骤，对应段落标注 `[!]`
